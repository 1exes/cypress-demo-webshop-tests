# cypress-demo-webshop-tests

E2E-Tests für den Tricentis Demo Webshop mit Cypress.

## Tech Stack
- Cypress
- JavaScript

## Features
- Vollständige E2E-Testabdeckung für den Demo-Webshop
- Warenkorb-Tests, Login, Checkout
- CI/CD-Integration

## Getting Started
```bash
npm install
npx cypress open
npx cypress run
```
